## 🚀 Featured Projects

### 🎮 DreadScape

A game where you take control of a pilot aboard a rusty land rover attempting to maneuver alone through a canyon across an alien planet. You're doing this to analyze the planet's habitability for an oligarchical organization back home, but as you traverse this destitute world a question constantly lingers at the back of your head... are you *really* alone out there?

I created this project using pygame in December 2024 as an introduction to understanding how game architecture and generic software dev practices are applied. This game was inspired by David Szymanski's indie horror game _Iron Lung_, however the story, art and audio design as well as all code in this project was created by me.<br><br>


### 🤖 AIIRA (Demo)

A locally-run Retrieval-Augmented Generation (RAG) pipeline that creates incident response plans using private reference documents and an open-source LLM. This is the functional baseline for a larger project that later deploys the same pipeline inside a confidential VM (TEE) for improved data privacy.

I worked on this project in a team of 6 people for a hackathon, based on a challenge of understanding TEEs as well as their application in order to assist as an AI tutor for students.<br><br>

### ☁️ Amusement Park Cloud System

A POC Streamlit app designed for amusement park guests to view information about ride congestions and queue-time prediction. Uses an agent-based simulation for crowd data, and is designed for GCP cloud integration as a demonstration for public deployment and management using cloud services.

The current version can run locally and was created as a display for how future cloud deployment may look.<br><br>
